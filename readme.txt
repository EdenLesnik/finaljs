SpaceWeb Portfolio Site
Welcome to SpaceWeb, a portfolio website showcasing various landing page projects created with HTML, CSS, and SASS.

Project Overview
SpaceWeb serves as a platform to display a collection of landing page projects developed during web development studies. Each landing page project is designed to demonstrate skills in HTML, CSS, and SASS.

Installation
Clone the repository:

bash
Copy code
git clone https://github.com/eden-lesnik/SpaceWeb.git
Navigate to the project directory:

bash
Copy code
cd SpaceWeb
Open the desired landing page project folder to view its files.

Usage
To view a specific landing page project, open the corresponding folder and explore the HTML, CSS, and SASS files. You can also open the index.html file in a web browser to see the landing page in action.

Contributions
Contributions are welcome! If you'd like to contribute to SpaceWeb by adding new landing page projects or improving existing ones, feel free to fork the repository and submit a pull request.
